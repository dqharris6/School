to compile:
- navigate to directory where CodeWriter.java, CullenDaltonParser.java, and 
VMCompiler.java are
- run this> javac *.java
- program should compile without errors

to run:
- run this> java VMCompiler.java filename.vm
- change filename.vm to one of the 5 test cases
- open CPUEmulator.sh
- run all test cases, comparing new .asm files 
- all should pass
